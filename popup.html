About

This extension calculates and displays the adjusted time for videos based on playback speed.

Version: 1.2

Help

Having trouble? Here's how to use this extension:

  • Change settings under the "Settings" tab.
  • Use the "Test Settings" button to preview changes.
  • Reset settings using the "Reset to Defaults" button.
